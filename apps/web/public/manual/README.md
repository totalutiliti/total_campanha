# Capturas de tela do Manual

Solte aqui as imagens (`.png`) com **exatamente** estes nomes. Cada arquivo preenche
um slot `<Figura>` no manual (`apps/web/src/app/(auth)/manual/conteudo.tsx`).
Enquanto o arquivo não existir, o slot mostra um placeholder "Captura de tela em breve" —
ou seja, o manual já funciona sem as imagens.

| Arquivo                        | Tela                                             |
| ------------------------------ | ------------------------------------------------ |
| `inicio-1.png`                 | Aba Início (roteiro "Comece por aqui" + atalhos) |
| `conexoes-1.png`               | Conexões (cartões WhatsApp e e-mail)             |
| `conexoes-whatsapp-wizard.png` | Assistente do WhatsApp (4 passos)                |
| `conexoes-email-dns.png`       | Registros DNS do e-mail                          |
| `contatos-1.png`               | Lista de contatos                                |
| `contatos-importar.png`        | Importação de planilha                           |
| `grupos-1.png`                 | Lista de grupos                                  |
| `grupos-novo.png`              | Construtor de regras de grupo                    |
| `mensagens-1.png`              | Lista de mensagens                               |
| `mensagens-nova.png`           | Formulário de nova mensagem                      |
| `campanhas-1.png`              | Lista de campanhas                               |
| `campanhas-nova.png`           | Nova campanha                                    |
| `campanhas-resultados.png`     | Painel de resultados                             |
| `respostas-1.png`              | Caixa de respostas                               |
| `plano-1.png`                  | Plano / cobrança                                 |
| `conta-1.png`                  | Minha conta                                      |

**Dica:** capture em tela cheia, tema claro, largura ~1280px. Recomenda-se recortar
só a área útil (sem a sidebar) para o print não competir com o layout do manual.
